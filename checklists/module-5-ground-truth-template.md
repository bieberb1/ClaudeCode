# Ground Truth Document Template

A Ground Truth Document is your single authoritative reference for a storytelling project. Every verified claim goes here. Every output traces back to it. If a fact isn't in the Ground Truth, it doesn't go in the output.

## What does NOT go in this document

- Claims you found but have not verified
- Statistics cited in secondary sources without tracing to the primary
- Figures whose source you cannot identify
- LLM-generated numbers that were not confirmed against a named database or study

---

## Use this document

Once your Ground Truth Document has five or more verified entries, you are ready for Stage 3 (Narrate). Use the Claude prompt from the Pipeline Builder playground to generate a first-draft Story Spine for each audience, with instructions to draw evidence only from this document.
